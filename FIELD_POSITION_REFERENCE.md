## Measurement Tips

### Using Adobe Reader:
1. Open Tools → Measure
2. Click on the start point (top-left of input field)
3. Drag to end point (bottom-right of input field)
4. Read coordinates from the measurement dialog
5. Record X, Y, Width values

### Units:
- All coordinates in **PDF Points** (1 point = 1/72 inch)
- If your tool shows mm, convert: mm × 2.834646 = points
- If your tool shows inches, convert: inches × 72 = points

### Width Measurement:
- Measure the width of each input box
- This prevents text from overflowing
- Especially important for longer fields (names, addresses)

### Recording:
- Write measurements clearly with units
- Take photos of your measurements
- Keep a backup copy of your recorded values

## Template for Updated submit_fpdi.php

Once you have all coordinates, format them like this in `submit_fpdi.php`:

```php
// PAGE 1 - IDENTIFICATION SECTION
addTextToPdf($pdf, $fullName, 50, 165, 500);           // Nome
addTextToPdf($pdf, $birthDate, 50, 190, 200);          // Nascido em
addTextToPdf($pdf, $nationality, 310, 190, 200);       // Nacionalidade
// ... (continue for all fields)

// PAGE 2 - FAMILY SECTION
addTextToPdf($pdf, $fatherName, 50, 80, 500);          // Pai - Nome
addTextToPdf($pdf, $fatherPhone, 50, 105, 200);        // Telemóvel Pai
// ... (continue for all fields)
```

### Function Signature:
```php
addTextToPdf(
    $pdf,           // PDF object
    $text,          // Text to display
    $x,             // X coordinate (distance from left)
    $y,             // Y coordinate (distance from top)
    $width,         // Width of text area
    'L',            // Alignment: 'L'eft, 'C'enter, 'R'ight
    10              // Font size (optional, default 10)
);
```
